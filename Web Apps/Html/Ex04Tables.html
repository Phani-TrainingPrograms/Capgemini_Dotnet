Tables

Tables are used to present the content in a tabular format. It could be the data extracted from a REST API getting its data from a Database. Most of the time, the data shall be dynamic and shall populate the data dynamically(At the runtime).

Points to Remember:

  • The Table tag is used to create the table <table>
  • It shall have 2 important Tags: tr and td which represents row of the table and the data in that column of the row respectively
  • Optionally U can use th tag for creating headings to the table.
  • Use the border attribute to set the border to the table. Alternatively, you can use the styles to achieve the same.
  • U can also use optional tags like tfoot to present footer for the table
  • tr and td have span(col and row) feature which allows the respective tags to span across the rows and columns.

Images in HTML Pages

Images are attached in HTML Docs using <img> Tag. Its src attribute is used to set the image address. Preferably using the relative path. Other attributes like Width and Height could be set. If an image could not be loaded for various reasons, U can provide alt attribute and set a string value as image information. It is recommended to create images folder and place all the required images for the website in it and refer it relatively. You should be aware of the copyright issues when U R using 3rd party images.

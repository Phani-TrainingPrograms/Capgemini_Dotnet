About Forms

Forms allow User inputs inside UR web pages. Forms work with server side Applications to store, process and render the data from the server. We can use server side technologies like ASP, JSP, ASP.NET, PHP, EXPRESS.JS to process the data sent to it and handle the persistance part of it.

Forms send its data to the server in 2 ways: GET and POST. Get sends the information thru' URL as QUERYSTRINGS. It is not safe as it is viewable by any users.
POST is the most secure way of sending Information from the Web page to the server as data is sent thru' body. Information to the server is posted as HTTP-REQUEST and obtain a HTTP-RESPONSE to that request.
Forms use Input tags for taking inputs from the user. there are other tags like select, radio to display drop down lists and radio buttons.

Points to remember

  • Form tag is used to create the Form and end it with End tag.
  • action attribute is used in Server side programming to define the page or the file that processes these inputs
  • input tags can be processed using Client side programming using JS
  • Every input element can be identified by id attribute which is used to refer it programmatically
  • We use DOM to manipulate the content of the HTML using JS
  • DOM is the virtual Tree View structure of the HTML document.
  • JS is used to process the DOM tree in the browser.
